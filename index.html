Html Order List

  1. Open your web browser.
  2. Visit the Official Notepad++ Website:
    Go to https://notepad-plus-plus.org/.
  3. Go to the Download Section on left side.
  4. Select latest version or what you want.
  5. Select the Correct installer either 32-bit or 64-bit version.
  6. Click on the download link and save the installer file.
  7. Locate and run the installer file. Where you your downloads are.
  8. Simply select next if it is your first time.
  9. Finish
  10. Congrutulation! You install Notepad++
    Open Notepad++ from the desktop shortcut or Start menu.
